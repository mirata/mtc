### Marathon VR

All the source code and customisations to making gzdoom behave like Marathon, including such things as:
- Platforms
- Lights
- Switches
- Terminals
- Polygon triggers
- Enemy behaviour
- Sprites/Textures
- Level goal logic
- Hud

## Build
`.\build.ps`    

If on non-windows, simply zip the `src` folder contents and rename as `.pk3`
